# 今天吃啥？

个人食谱计划和采购清单应用。

## 本地运行

```bash
npm install
npm run dev
```

## 部署到 GitHub Pages

推送到 GitHub 仓库的 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

当前仓库地址：

```text
https://github.com/Jackiehill-ff/What-food-today
```

发布地址通常是：

```text
https://jackiehill-ff.github.io/What-food-today/
```
